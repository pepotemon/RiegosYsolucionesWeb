# Segundo Cerebro - Riegos y Soluciones Agricolas del Norte

Este documento es el centro interno del proyecto. No debe publicarse como contenido visible del sitio.

## Objetivo Del Proyecto

Crear una pagina web comercial para **Riegos y Soluciones Agricolas del Norte**, enfocada en generar confianza, mostrar experiencia y captar solicitudes de cotizacion.

La frase guia del proyecto:

> No estamos construyendo una pagina web. Estamos construyendo el vendedor digital de la empresa.

## Estado Actual

- Plan elegido: Web Comercial.
- Cliente: colega/amiga.
- Prioridad: entregar una web profesional y reutilizable como plantilla futura.
- Stack implementado: Next.js, TypeScript, Tailwind CSS, Framer Motion y Lucide React.
- Estado tecnico: sitio construido con rutas principales, rutas dinamicas, datos locales, SEO base, sitemap y robots.

## Sitio Publico

Paginas incluidas:

- Home.
- Servicios.
- Servicio individual.
- Proyectos.
- Proyecto individual.
- Catalogo comercial.
- Blog.
- Articulo individual.
- Nosotros.
- Contacto.

## Reglas Del Proyecto

- Primero conversion, despues decoracion.
- Cada seccion debe explicar que hace la empresa, por que confiar, que problema resuelve y como solicitar cotizacion.
- Evitar texto generico, diseno recargado, paginas sin CTA, servicios mal explicados y formularios demasiado simples.
- Mantener claridad, profesionalismo, orden, velocidad y buena experiencia movil.

## No Incluye En Esta Fase

- CRM completo.
- Panel administrativo avanzado.
- Automatizacion real por WhatsApp API.
- Portal privado de clientes.
- Portal de tecnicos.
- Pasarela de pagos.
- E-commerce.

## Futuro

La arquitectura queda preparada para conectar Firebase, Firestore, Firebase Storage, un CMS, CRM, WhatsApp API, agenda de visitas tecnicas, portal privado, modulo de cotizaciones e IA para consultas frecuentes.
