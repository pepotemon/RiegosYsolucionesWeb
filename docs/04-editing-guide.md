# Guia Rapida Para Editar Contenido

El sitio usa datos locales para que sea facil editar sin tocar demasiados componentes.

## Empresa Y Contacto

Archivo:

- `lib/site.ts`

Editar aqui:

- Nombre.
- Descripcion.
- URL.
- Telefono.
- WhatsApp.
- Correo.
- Direccion.
- Redes sociales.

## Servicios

Archivo:

- `data/services.ts`

Cada servicio controla:

- Pagina de servicio individual.
- Card en Home.
- Card en `/servicios`.
- Preguntas frecuentes.
- Proyectos relacionados.

## Proyectos

Archivo:

- `data/projects.ts`

Cada proyecto controla:

- Card de portafolio.
- Pagina individual.
- Galeria.
- Problema.
- Solucion.
- Resultado.

## Productos

Archivo:

- `data/products.ts`

Cada producto controla:

- Catalogo.
- Categoria.
- Marca.
- Caracteristicas.
- Boton de WhatsApp con mensaje prellenado.

## Blog

Archivo:

- `data/blog.ts`

Cada articulo controla:

- Card del blog.
- Pagina individual.
- Metadata SEO.

## Nosotros

Archivo:

- `data/company.ts`

Editar aqui:

- Historia.
- Mision.
- Vision.
- Valores.
- Experiencia.
- Cobertura.
- Equipo tecnico.
